## What this repo contains

Advanced SQL work completed as part of a structured 8-month BA/DA analytics roadmap. This repository demonstrates applied SQL and data analytics capabilities through progressively complex work.

Content includes:

- Window functions: ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD, NTILE, running totals  
- CTEs and subquery refactoring  
- Query optimisation techniques: EXPLAIN plans, index usage, reducing full table scans  
- Database design: normalization and ERD documentation  
- LeetCode SQL 50 solutions (30+ problems, Easy to Medium)  
- Analytical queries using public datasets (ABS and Stats NZ)  
- DA track: SQL applied on pandas DataFrames using pandasql and DuckDB
